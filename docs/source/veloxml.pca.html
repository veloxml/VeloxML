class veloxml.pca.pca.PCA(n_components=5)[source]

Bases: UnsupervisedEstimatorBase

Principal Component Analysis (PCA) model.

This class implements Principal Component Analysis (PCA), a dimensionality reduction technique that transforms data into a lower-dimensional space while preserving the maximum variance.

The number of principal components to retain.

Type:

Parameters:

Example

>>> model = PCA(n_components=3)
>>> model.fit(X_train)
>>> transformed_data = model.transform(X_train)
>>> components = model.get_components()
