Elastic Net Regression model with combined L1 and L2 regularization.

This class implements Elastic Net Regression, which combines L1 (Lasso) and L2 (Ridge) regularization to balance sparsity and shrinkage of coefficients.

  • mode (Literal["FISTA", "ADMM"], optional) – The solver mode used for optimization. Defaults to “FISTA”. Must be one of [“FISTA”, “ADMM”].

  • lambda_l1 (float, optional) – The L1 regularization strength. Defaults to 1.0.

  • lambda_l2 (float, optional) – The L2 regularization strength. Defaults to 1.0.

  • penalize_bias (bool, optional) – If True, applies regularization to the bias term. Defaults to False.
